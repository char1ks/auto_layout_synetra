# 🚀 ГИБРИДНАЯ СИСТЕМА: LLaVA + SearchDet + SAM2

## ✨ **Особенности новой системы:**

### **🎯 Решение проблемы отсутствующих элементов**
- **LLaVA**: Понимает контекст и общие дефекты
- **SearchDet**: Находит отсутствующие элементы по примерам  
- **SAM2**: Точная сегментация найденных проблем

---

## 📁 **ПОДГОТОВКА ПРИМЕРОВ (КРИТИЧНО!)**

### **Структура папок:**
```
examples/
├── positive/          # ПРАВИЛЬНЫЕ элементы
│   ├── good_contact_1.jpg
│   ├── good_contact_2.jpg
│   └── good_contact_3.jpg
└── negative/          # НЕПРАВИЛЬНЫЕ/отсутствующие элементы  
    ├── missing_contact_1.jpg
    ├── empty_socket_1.jpg
    └── damaged_pin_1.jpg
```

### **Правила создания примеров:**

#### **POSITIVE (положительные)** - как ДОЛЖНО быть:
- ✅ Разъемы С контактами
- ✅ Провода С изоляцией  
- ✅ Полные сборки
- ✅ Правильное расположение элементов

#### **NEGATIVE (отрицательные)** - что НЕ ДОЛЖНО быть:
- ❌ Пустые разъемы БЕЗ контактов
- ❌ Голые провода БЕЗ изоляции
- ❌ Отсутствующие компоненты
- ❌ Неправильное расположение

### **Пример для вашего изображения:**

**POSITIVE папка** (как должно выглядеть):
```bash
mkdir -p examples/positive
# Добавьте изображения разъемов С медными контактами
# Изображения полных, правильных сборок
```

**NEGATIVE папка** (что является проблемой):
```bash
mkdir -p examples/negative  
# Добавьте изображения пустых разъемов БЕЗ контактов
# Изображения с отсутствующими элементами
```

## 🛠️ **УСТАНОВКА ГИБРИДНОЙ СИСТЕМЫ**

### **Шаг 1: Установка зависимостей**

```bash
# Базовые зависимости
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
pip install -r requirements_llava_sam2.txt

# Дополнительные для SearchDet
pip install faiss-gpu  # или faiss-cpu для CPU
pip install scikit-learn
pip install selenium  # для веб-скрапинга (опционально)
```

### **Шаг 2: Установка моделей**

```bash
# LLaVA
git clone https://github.com/haotian-liu/LLaVA.git
cd LLaVA && pip install -e . && cd ..

# SAM2  
git clone https://github.com/facebookresearch/segment-anything-2.git
cd segment-anything-2 && pip install -e . && cd ..

# SearchDet уже в папке searchdet-main/
```

### **Шаг 3: Скачивание весов SAM-HQ для SearchDet**

```bash
# Создаем папку для весов SearchDet
mkdir -p searchdet-main/sam-hq/pretrained_checkpoint

# Скачиваем веса SAM-HQ
cd searchdet-main/sam-hq/pretrained_checkpoint
wget https://huggingface.co/lkeab/hq-sam/resolve/main/sam_hq_vit_l.pth
cd ../../..
```

## 🚀 **ЗАПУСК ГИБРИДНОЙ СИСТЕМЫ**

### **Базовый запуск:**
```bash

    --image 001.png \
    --positive examples/positive \
    --negative examples/negative \
    --output output \
    --model detailed
```

## 🔧 **НАСТРОЙКА ПАРАМЕТРОВ**

### **В файле `hybrid_searchdet_pipeline.py` можно изменить:**

```python
# Пороги для SearchDet
similarity_threshold=0.4  # Чем ниже, тем чувствительнее к отсутствующим элементам

# Параметры SAM
points_per_side=32       # Больше = детальнее, но медленнее
pred_iou_thresh=0.8      # Качество масок
```

## 🆚 **ПРЕИМУЩЕСТВА ГИБРИДНОГО ПОДХОДА**

| Проблема | Старый подход | Гибридный подход |
|----------|---------------|------------------|
| Отсутствующие провода | ❌ Не находит | ✅ SearchDet находит |
| Контекстный анализ | ❌ Только SAM | ✅ LLaVA понимает |
| Точная сегментация | ⚠️ Приблизительно | ✅ SAM2 с координатами |
| Примеры обучения | ❌ Нет | ✅ Few-shot learning |
